### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `Visual Studio Code(VScode)`
- `Navicat`

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**学生功能：**
1. 登录
2. 注册
3. 首页
4. 学生问题
5. 老师回答
6. 老师信息
7. 交流区
8. 个人中心
9. 我的发布
10. 我的收藏
11. 后台管理



**教师功能：**
1. 登录
2. 注册
3. 个人中心（修改密码、个人信息）
4. 老师回答管理
5. 老师信息管理
6. 关注列表管理
7. 我的收藏管理



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 学生管理
4. 老师管理
5. 科目类型管理
6. 学生问题管理
7. 老师回答管理
8. 老师信息管理
9. 关注列表管理
10. 交流区
11. 系统管理（轮播图管理）
